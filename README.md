# NAS File Manager (Tauri)

基于 **Tauri 2 + React + Rust** 的跨平台 NAS 文件管理器原型，目标是提供接近 macOS Finder 的连接与浏览体验，并支持 SMB/WebDAV 多协议统一抽象。

## 功能现状

- 连接管理：新增/编辑/删除连接，一键进入文件浏览页
- WebDAV：
  - PROPFIND 列目录（解析 XML）
  - 支持自签名 HTTPS 证书（局域网常见）
  - 支持中文路径、目录结尾 `/` 兼容性
  - 支持连接 URL 自带“基路径”，并映射为应用内根目录 `/`
  - **全新架构：内置本地代理服务器 (`axum`)，绕过 CORS 和 Basic Auth，实现流式多媒体播放**
- SMB：
  - 基于 `smb-rs` 实现（当前与部分 NAS 的 NTLM 兼容性存在问题，后续计划替换为 Samba `libsmbclient`）
- 局域网发现：
  - mDNS `_smb._tcp` 扫描，快速发现设备并填入连接表单
- 状态与安全：
  - 支持“记住密码”并在再次编辑时屏蔽密码明文显示，保护隐私。
- 文件浏览与交互：
  - **网格视图与缩略图**：支持切换至网格/大图标模式，图片文件自动通过本地代理生成缩略图，其他文件（音视频、文档等）展示专属精美图标，完美复刻桌面端资源管理器体验。
  - **响应式面包屑导航**：支持超长路径自动折叠 (`...`)，并在窄屏下自动切换为两行工具栏布局。
  - 排序（默认按时间倒序；支持名称/时间/大小，目录优先，**名称排序支持自然数排序**：如 1, 2, 10）
  - 新建文件夹 / 重命名 / 删除（权限不足会提示）
  - **全触控与响应式自适应**：
    - 支持长按触发文件操作菜单（移动端/平板体验）。
    - 窄屏模式下，左侧边栏自动隐藏并转化为抽屉（Drawer）汉堡菜单。
    - 超长文件名自动截断，窄窗口三点按钮绝对防溢出。
- 下载与传输：
  - 支持 Rust 底层多线程断点续传下载
  - 独立的全局传输管理面板，支持暂停、恢复、取消
- 多媒体预览 (沉浸式暗黑 UI)：
  - **图片浏览**：全局沉浸式半透明模糊遮罩，支持无缝左右点击或**滑动翻页**
  - **视频播放 (极致交互)**：
    - 完全自定义极简视频控件，摒弃系统强制遮罩
    - **支持触控手势**：左右滑动快进退，上下滑动调节音量
    - **长按触发 2.0x 倍速播放**
    - 支持最高 2.0x 实时倍速调节，以及悬浮音量条与音量记忆功能

## 开发

```bash
npm install
npm run tauri dev
```

## WebDAV 连接建议

- 群晖 WebDAV Server 常见默认端口：
  - HTTP: `5005`
  - HTTPS: `5006`
- 连接 URL 可以直接指向某个目录作为“根”，例如：
  - `https://192.168.1.10:5006/临时空间`
